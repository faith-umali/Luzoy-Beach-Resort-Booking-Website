# 🌊 Luzoy Beach Resort Booking Website
A booking website developed for Luzoy Beach Resort using HTML, CSS, and JavaScript. The website provides accommodation details, rates, amenities, photo galleries, and an online booking form for visitors.

## Overview

The Luzoy Beach Resort Booking Website is a front-end web development project created for our BSCS Web Development course. It was developed to provide visitors with an organized platform where they can view resort information, explore available accommodations, check rates and amenities, browse photo galleries, and submit booking requests.

## Features

- Home page with a modern landing section
- Room and cottage gallery
- Accommodation information
- Rates and amenities
- Image preview modal
- Online booking form
- About and Contact sections
- User-friendly interface

## Built With

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts (Poppins)

## Project Purpose

This project was developed as part of our 2nd-year BSCS Web Development. The main objective was to apply front-end web development concepts by creating a functional and organized booking website for a beach resort.

## My Contribution

As the team leader, I was primarily responsible for the website's overall design, page layout, front-end development, and implementation of the website's interactive features while working together with my group members throughout the project.
